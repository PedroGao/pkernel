# 库函数与调试打印函数

c 语言的标准库均不能使用，因此需要自己去实现字符串和 debug 相关的库，cool~。

## 字符串

字符串相关的操作，如比较、拼接，获取长度等，主要考察 c 指针的用法，指针真的是个迷人的好东西。

## 打印函数

如何实现可变长参数，主要是根据参数从右到左来如栈。

## 反汇编

这节内容巨多，很多操作系统寄存器的知识，不会汇编的真的是摸瞎。

可通过反汇编来查看内核代码，如下：

``` sh
objdump -M intel -d hx_kernel
```

## 调试

添加 `scripts/gdbinit` 文件，然后执行:

``` sh
make debug
```
